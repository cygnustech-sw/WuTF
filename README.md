# WuTF – Windows Update Toolkit Friend

**WuTF (Windows Update Toolkit Friend)** is a lightweight PowerShell toolkit that helps Windows administrators quickly triage and address common Windows Update and WSUS issues.

The toolkit provides a simple, menu-driven workflow for:

- Checking Windows Update–related services
- Clearing the Windows Update cache
- Triggering a scan from WSUS/Windows Update
- Running system health checks (SFC/DISM)
- Producing timestamped logs for troubleshooting
